A-view . It is simple, user-friendly video conferencing software, which provides a great opportunity to a teacher to teach in a live interactive mode to various geographical locations across India. A-View classroom provides the opportunity to connect several universities together and create a virtual world for students. It also acts a knowledge café where students can discuss/chat about lecture after the live class.

One challenge was deciding how to layout the 3×2 grid of Services for mobile users. I ultimately created an accordion that would display more information upon clicking the heading. This was my first attempt at writing my own jQuery code, and it turned out to be a great success.
